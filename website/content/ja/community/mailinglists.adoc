= メーリングリスト

メーリングリストは、FreeBSD コミュニティの主要なコミュニケーションチャンネルで、 幅広い話題をカバーしています。

== メーリングリストのアーカイブ

メーリングリストのアーカイブを https://www.FreeBSD.org/[www.FreeBSD.org] にて link:../../search/#mailinglists[検索]したり、 https://www.FreeBSD.org/mail/[閲覧]できます。 Mailman ウェブインターフェースから https://lists.FreeBSD.org/mailman/listinfo[閲覧]することもできます。

== 英語のメーリングリスト

英語のメーリングリストは、 link:{handbook}#eresources-mail[FreeBSD ハンドブックにまとめられています。]

== 英語以外のメーリングリスト
